# Inter Font Files

Please place your Inter OTF font files in this directory with the following naming:

- `Inter-Regular.otf` (weight: 400)
- `Inter-Medium.otf` (weight: 500)
- `Inter-SemiBold.otf` (weight: 600)
- `Inter-Bold.otf` (weight: 700)

You can download Inter from: https://rsms.me/inter/

Make sure to use the OTF (OpenType Font) format files.
